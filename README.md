# scientific-modeling-lab
Contains midterm/final source codes that are associated with project N-body simulations in cosmology.

### Content of folders
*G2_galaxy*: configuration and parameter files for $\text{\texttt{GADGET-4}}$ example simulation $\text{\texttt{G2-galaxy}}$, simulation snapshots I made, jupyter notebook for plotting data\
\
*DM_L50_N128*: configuration and parameter files for $\text{\texttt{GADGET-4}}$ example simulation $\text{\texttt{DM-L50-N128}}$, simulation snapshots I made, jupyter notebook for plotting data (including relative particle density and its Fourier transform)\
\
*power_spectrum*: jupyter notebook for determining and plotting non-linear power spectra corresponding to $\text{\texttt{DM-L50-N128}}$ snapshots\
\
*midterm_report_source*: all files needed to compile the midterm report\
\
*final_report_source*: all files needed to compile the final report\
\
*power_spectrum_cpp_source*: the source files of power spectrum calculating $\text{\texttt{c++}}$ code\
\
(all $\text{\texttt{.hdf5}}$ snapshots and $\text{\texttt{.pdf}}$ images are saved using $\text{\texttt{Git LFS}}$)
